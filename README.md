# AOSP Gesture Bar for Oxygen OS

Simple module to give Oxygen OS 10 users the AOSP Gesture Bar (in case you're not a fan of the wide pill that Oneplus uses)

## Requirements
- Android 10 only
- Magisk 19+

## Installation
1. Flash this module.
2. Reboot.
3. Enjoy!
